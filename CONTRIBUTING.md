# Contributing

This catalog only works if it's current and real. Two ways to help:

## 1. Suggest a program (easiest)

[Open an "Add a program" issue →](https://github.com/landedjobs/ai-fellowships-and-residencies/issues/new?template=add-program.yml) with the name, official link, and whatever you know (stipend, deadline, who it's for). New/hot programs are especially welcome for the **🔥 Popular & New** section.

## 2. Open a PR

Add or update a program block, matching the existing format:

```
### Program Name — Host Org

| | |
|---|---|
| **What** | type · focus |
| **For** | eligibility / career stage |
| **Deal** | duration · stipend · location |
| **2026 cycle** | window/deadline — mark "(estimated from past cycles)" if unconfirmed |

One-to-two sentence description. **[Apply / details →](official-url)**
```

Put it in the right category (Frontier labs / AI safety / Academic / Startup & VC / Policy & government / Career-switcher & international), or in **🔥 Popular & New** if it's genuinely buzzy right now.

## Ground rules

- **Real and verifiable only** — link the official page. No aggregator-only entries if an official page exists.
- **Flag dead ones.** If a program is discontinued or on hiatus, say so — we keep those in the collapsed section rather than deleting, so people stop chasing them.
- **Deadlines rot fast.** If you spot a stale date, a one-line PR fixing it is a real contribution.

Thanks for keeping it accurate. 🙏
