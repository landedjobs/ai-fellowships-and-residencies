| | |
|---|---|
| **What** | Research fellowship · Empirical AI safety on LLMs — interpretability, control, scalable oversight, adversarial robustness, model welfare, AI security |

### Microsoft Research AI Residency — Microsoft Research

12-month RSE-style research training program (Redmond, Cambridge UK, Montreal), historically aimed at career switchers transitioning into ML research. Effectively legacy — the last detailed blog post dates to March 2020, and some Microsoft Research labs now hire ML R&S roles directly instead. [Legacy page →](https://www.microsoft.com/en-us/research/blog/ai-residency-gaining-skills-in-machine-learning-through-mentorship-and-real-world-applications/)
